〜概要〜
指定したフォルダを監視して、binファイルを発見し次第、指定ドライブに転送するツールです。

〜インストール・アンインストールについて〜
任意の場所にフォルダごと配置してください。
アンインストールはフォルダごと削除してください。
レジストリは使用していません。

〜アップデートについて〜
本フォルダの中身をすべて上書きしてください。
設定ファイルは含まれておりませんので、バージョンアップ後
も同じ設定でお使いいただけます。

〜使い方〜
初回起動時、設定ツールのConfig.exeを起動してください。
監視対象フォルダ、通知音の設定を行いましたら右の設定保存ボタンを押してから
設定ツールを終了してください。以降のmbed AutoTransfer.exe起動時の設定として
読み込まれます。

本体のmbed AutoTransfer.exeを起動すると監視が始まります。
　※注意：起動前に監視フォルダにbinファイルはない状態にしてください。ドライブを選択する
　　　間もなく転送してしまいます。
mbedのドライブを選択してください。
この状態でオンラインIDEからbinファイルを監視フォルダへダウンロードすると
自動でmbedのドライブに転送します。

〜音源について〜
通知音の音源はゼロワンサウンドアース様の素材ID b_112を利用しております。
http://www.01earth.net/sound/
